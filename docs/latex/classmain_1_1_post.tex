\begin{DoxyVerb}This class create a table post with various column to store infromation about post.
Table column:
user_id : Store user id who posted post
user_name : Store user Name who posted post
Below column are related to flat information
state : Stroe flat state,
city  : Stroe flat city,
area  : Stroe flat area,
date_posted  : Stroe the date in which information posted about flat,
pincode   : Stroe flat area pincode
rent       : Stroe flat rent 
'Below column are realated to facilities'
furnishing_type,
washing_machine,
bedrooms,
ac,
almirah,
bed,
geyser,
gas_stove,
sofa,
availability,
landloard_nature,
comments,
flat_rating,
flat_rating,
picture1 : store picture about flat
picture2 : store picture about flat
picture3 : store picture about flat
picture4 : store picture about flat
picture5 : store picture about flat
picture1 : store picture about flat
This table stores all the data related to the posts done by User and inherits db.Model class
id is primary key for each post

stores name with extension of image \end{DoxyVerb}
