Detailed Description

 This class is a 'form', it inherits class FlaskForm and provides a structure for form which accepts all the details regarding a post
 BooleanField, which is a checkbox to get whether the respective amenity is provided or not 
 variable landloard_nature stores input text as string, that is, nature of landlord according to the user 
 variable neighbours_nature stores input text as string, that is, nature of neighbours according to the user 
 variable comments stores input text as string, that is, additional comment regarding the place 
 variable nearby stores input text as string, that is, nearby areas 
 variable flat_rating stores overall rating of the place as an integer from the point of user who is posting  
 Filefield which takes file as input of the given formats 
 SubmitField to submit the form that has all the details regarding a post
